## :closed_book: Project Directory Structure
```bash

├── demoapp/          # App folder

│ │ ├── tests.py      # Test fo the app
│ │ ├── urls.py       # Define the URL routing for the app
└── └── views.py      # User define function called when get client request URL 
```
